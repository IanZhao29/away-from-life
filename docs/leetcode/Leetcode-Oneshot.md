# Leetcode 秒杀题目合集

## 2021/12/12 [709. 转换成小写字母](https://leetcode-cn.com/problems/to-lower-case/)

给你一个字符串 `s` ，将该字符串中的大写字母转换成相同的小写字母，返回新的字符串。

 ### 示例

```away
输入：s = "Hello"
输出："hello"
```

一行搞定流重出江湖

```javascript
/**
 * @param {string} s
 * @return {string}
 */
var toLowerCase = function(s) {
    return s.toLocaleLowerCase();
};
```
